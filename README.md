# Bike Sales Excel Dashboard

## Overview

This project is an interactive Bike Sales Dashboard built in Microsoft Excel to understand customer purchasing behavior and generate useful insights. The dataset was cleaned and organized before being analyzed using pivot tables and visualized with charts.

## Objectives

* Understand customer buying patterns
* Identify factors that influence bike purchases
* Present insights in a clear and interactive way

## Tools and Techniques

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Charts and Graphs
* Slicers and Filters

## Key Insights

* Customers with higher income are more likely to purchase bikes
* Middle-aged individuals show higher purchase rates
* Married customers tend to buy more bikes compared to single customers
* Region and education level also influence buying decisions

## Features

* Interactive dashboard using slicers
* Dynamic charts for better understanding
* Clean and structured dataset
* Simple and easy-to-read layout
